# الحكيم الذكي Pro - Smart Medical Sage

نظام طبي متقدم مدعوم بالذكاء الاصطناعي للتشخيص والمتابعة الطبية.

## التشغيل المحلي

1. تثبيت التبعيات:
   `npm install`
2. تشغيل المشروع:
   `npm run dev`

## التقنيات المستخدمة
- React 18
- Tailwind CSS 4
- Google Gemini API
- Supabase (PostgreSQL)
- Motion (Animations)
