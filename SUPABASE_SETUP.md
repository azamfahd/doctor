# دليل ربط مشروع الحكيم برو مع Supabase

## ✅ ما تم إنجازه

تم ربط مشروع الحكيم برو بنجاح مع قاعدة بيانات Supabase. المشروع الآن يدعم:

### 1. **التخزين السحابي**
- جميع السجلات الطبية محفوظة في Supabase
- الإعدادات محفوظة سحابياً وتتزامن تلقائياً
- لا حاجة للقلق بشأن فقدان البيانات

### 2. **المزامنة التلقائية**
- عند فتح البرنامج، يتم جلب جميع البيانات من السحابة
- أي تعديل يتم حفظه فوراً في قاعدة البيانات
- يمكنك الوصول للبيانات من أي جهاز

### 3. **الجداول المنشأة**
تم إنشاء جدولين رئيسيين:

#### جدول `records` (السجلات الطبية)
```
- id: معرف فريد للسجل
- name: اسم المريض
- age: العمر
- gender: الجنس
- symptoms: الأعراض
- vitals: المؤشرات الحيوية (JSON)
- image: صورة المريض (إن وجدت)
- diagnosis: التشخيص (JSON)
- chat_history: سجل الاستشارات (JSON)
- date: تاريخ الفحص
- status: حالة المريض (عادية/متابعة/عاجلة)
- created_at: تاريخ الإنشاء
```

#### جدول `settings` (الإعدادات)
```
- id: معرف الإعدادات
- center_name: اسم المركز الطبي
- doctor_name: اسم الطبيب
- personality: شخصية الذكاء الاصطناعي
- model: نموذج الذكاء الاصطناعي المستخدم
- deep_thinking: تفعيل التفكير العميق
- thinking_budget: ميزانية التفكير
- google_search: تفعيل البحث في جوجل
- theme: المظهر (فاتح/داكن)
- auto_save: الحفظ التلقائي
- voice_enabled: تفعيل الصوت
- voice_output_enabled: تفعيل إخراج الصوت
- profile_image: صورة الملف الشخصي
- api_key: مفتاح API
- updated_at: تاريخ آخر تحديث
```
